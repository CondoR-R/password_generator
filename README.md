# 🔐 Генератор паролей на Go

Простой и безопасный генератор случайных паролей с поддержкой копирования в буфер обмена.
Использует криптостойкий генератор `crypto/rand` и позволяет гибко настраивать состав пароля через аргументы командной строки.

## ✨ Возможности

- Генерация пароля заданной длины (по умолчанию 16 символов)

- Включение цифр (флаг `-d` / `-digits`)

- Включение специальных символов (флаг `-s` / `-symbols`)

- Копирование в буфер обмена (флаг `-c` / `-copy`)

- Получение информации о работе программы и возможных аргументах настройки
(флаг `-h` / `-help`)

- Понятный вывод в терминал с указанием использованных параметров

- Кросс-платформенность (Windows, macOS, Linux)

## 📦 Установка

### Вариант 1: Скачать готовый бинарник (рекомендуется)

1. Перейдите на [страницу релизов](https://github.com/CondoR-R/password_generator/releases).
2. Скачайте архив, соответствующий вашей операционной системе и архитектуре:
   - **Windows (64-bit):** `password-generator-windows-amd64.exe`
   - **macOS (Intel):** `password-generator-darwin-amd64`
   - **macOS (Apple Silicon):** `password-generator-darwin-arm`
   - **Linux (64-bit):** `password-generator-linux-amd64`
3. (Только для Linux/macOS) Сделайте файл исполняемым:
```bash
chmod +x password-generator
```
4. (Опционально) Переместите файл в директорию, находящуюся в `PATH`, например `/usr/local/bin`, чтобы запускать из любого места:
```bash
sudo mv password-generator /usr/local/bin/
```

#### Зависимости для работы с буфером обмена на Linux

Для корректного копирования пароля в буфер обмена на Linux требуется установить одну из утилит: `xclip` или `xsel`.

```bash
# Ubuntu/Debian
sudo apt install xclip

# Fedora
sudo dnf install xclip

# Arch Linux
sudo pacman -S xclip
```

На Windows и macOS дополнительные действия не нужны.

## Вариант 2: Сборка из исходников (требуется Go)

1. Убедитесь, что у вас установлен Go (версия 1.16 или новее).

2. Склонируйте репозиторий или скопируйте файлы проекта.

3. Соберите исполняемый файл:

```bash
go build -o password-generator
```

Для Linux потребуется установить `xclip` или `xsel` (для работы с буфером обмена):

```bash
# Ubuntu/Debian
sudo apt install xclip

# Fedora
sudo dnf install xclip

# Arch Linux
sudo pacman -S xclip
```

На Windows и macOS дополнительные действия не нужны.

## 🚀 Использование

Программа работает в терминале. Для запуска необходимо открыть терминал в директории с файлом.

```bash
# Linux/macOS
./password-generator [длина] [-d] [-digits] [-s] [-symbols] [-c] [-copy] [-h] [-help]

# Windows
.\password-generator [длина] [-d] [-digits] [-s] [-symbols] [-c] [-copy] [-h] [-help]
```

- длина – целое число, длина пароля (по умолчанию 16)

- `-d` или `-digits` – включить цифры в пароль

- `-s` или `-symbols` – включить специальные символы

- `-c` или `-copy` - включить копирование в буфер обмена

- `-h` или `-help` - получить информацию о работе программы и возможных аргументах настройки

Порядок аргументов не важен. Неизвестные флаги игнорируются.

### Примеры

#### Пароль по умолчанию (16 символов, только буквы):

```bash
$ ./password-generator
Длина пароля: 16
Использование цифр: выкл
Использование специальных символов: выкл
Сгенерированный пароль: tcFSaARqHhrvmhxH
Нажмите Enter для выхода...
```

#### Пароль длиной 20 символов с цифрами, копирование в буфер обмена:

```bash
$ ./password-generator 20 -digits -copy
Длина пароля: 20
Использование цифр: вкл
Использование специальных символов: выкл
Сгенерированный пароль: aB3dE5fG7hJ9kL1mN2pQ
Пароль скопирован в буфер обмена!
Нажмите Enter для выхода...
```

#### Пароль с цифрами и специальными символами:

```bash
$ ./password-generator -s -d 12
Длина пароля: 12
Использование цифр: вкл
Использование специальных символов: вкл
Сгенерированный пароль: K#8mP@2qR$5t
Нажмите Enter для выхода...
```

#### Если копирование не удалось (например, нет xclip на Linux):

```bash
$ ./password-generator -c
Длина пароля: 16
Использование цифр: выкл
Использование специальных символов: выкл
Сгенерированный пароль: tcFSaARqHhrvmhxH
Не удалось скопировать в буфер обмена: exec: "xclip": executable file not found in $PATH
Нажмите Enter для выхода...
```
